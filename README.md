# <strong>Installation</strong>
 - Install Biopython 
 
 ```
 pip install biopython
 ```
 - Clone git 
 
   ``` 
   cd /path/to/folder_you_want_to_store
   git clone https://github.com/Thanakron1997/Convert-FASTQ-to-FASTA.git
   cd Convert-FASTQ-to-FASTA
   ```
